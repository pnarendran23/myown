How do I register my device?
Registration screen appears when AlertApp starts first time. Enter 10 digit mobile number (as per school records). Enter the received OTP in validation screen.

Why Error "Invalid mobile number"?
Please make sure you enter the parent’s mobile number as given in the school. OTP is sent only to parent’s mobile number.

Can I register multiple devices for my child?
Yes. Enter the parent’s mobile number in the registration screen of the device (Android or iOS only). OTP will be sent only on the parent’s mobile number and that OTP is to be entered in the device you want to register.

I didn’t receive OTP
OTP is sent only to the parent’s mobile number as given in school records. If you did not receive OTP within two minutes you can request for resend of OTP.

What do these OTP related messages mean ? 

  • “OTP attempts exceeded, try Register again” For every registration an OTP will be sent to the parent’s mobile and same should be entered in validation screen. If wrong OTP is entered for three times application will prompt this message.
  • “OTP Limit exceeded. Contact school@groupten.com” If user attempts to register three consecutive times without successfully authenticating OTP, the app will prompt this message. The device is blocked in the system.
  • “Invalid OTP” If user enters wrong OTP the application will show this message, in this case user should request to resend OTP.

How do I change my mobile number in this application?

Make sure the new mobile number is updated in the school. Go to settings click on Mobile number Popup. Dialog will appear and allows the user to edit the mobile number and click on change button. You will be redirected to the registration screen.
